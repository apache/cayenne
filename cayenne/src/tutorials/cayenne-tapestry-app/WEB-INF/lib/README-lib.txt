
Copy cayenne.jar, ognl-*.jar and tapestry-*.jar into this directory. There is likely other jar files that are expected by 
Tapestry, such as bsf-*.jar, commons-digester-*.jar, commons-fileuplod-*.jar, javaassist-*.jar, etc. These files are
included in Tapestry distribution.

Optionally copy a JDBC driver used by the web application.
