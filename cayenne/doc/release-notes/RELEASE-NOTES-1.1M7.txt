Cayenne Release Notes 1.1 M7
Date: 06/14/2004
============================================

For the latest information go to the project web site at 
http://objectstyle.org/cayenne/

To browse individual bug reports, see 
http://objectstyle.org/jira/secure/BrowseProject.jspa?id=10000

Changes, Features And Improvements:

CAY-92	Shared namespace that spans multiple DataMaps within DataDomain.
CAY-36	Caching query results, including lookup data
- DataContext.performQueries(Collection, OperationObserver, Transaction) will no longer generate
  prefetch queries. Instead this is done upstream in performQuery(GenericSelectQuery). If you subclassed
  DataContext for whatever reason, check if this affects your subclass.
